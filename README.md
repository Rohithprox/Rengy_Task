# Fertinist Landing Page

React + Vite landing page for the Mythri Fertinest fertility treatments experience.

## What remains

- Fertility treatments landing page
- Fertinest footer
- Fertinist/Fertinest visual assets used by those sections

## Run locally

```bash
npm install --prefix client
npm run dev
```

The root `dev`, `build`, and `preview` scripts forward to the client app.
